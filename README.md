# 🐾 Filhos de 4 Patas - Pet Shop em Carapicuíba - SP

Bem-vindo(a) ao repositório do site **Filhos de 4 Patas**! Este projeto é uma Landing Page profissional, interativa e totalmente responsiva desenvolvida para um Pet Shop completo localizado em Carapicuíba - SP, oferecendo serviços e cuidados de alta qualidade para cães e gatos.

---

## ✨ Funcionalidades do Site

- 📱 **Design Responsivo:** A interface se adapta perfeitamente em dispositivos móveis (smartphones, tablets) e desktop.
- 🧼 **Apresentação de Serviços:** Seções detalhadas de Banho, Tosa, Veterinário, Hotel Pet, Creche e Transporte (Táxi Dog).
- 💬 **Agendamento Prático:** Links e botões de contato direcionados para otimizar os agendamentos dos clientes.
- 🎨 **Interface Premium:** Estilização moderna com Tailwind CSS e componentes acessíveis utilizando a biblioteca `shadcn/ui`.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias e ferramentas modernas:

- **[React](https://react.dev/):** Biblioteca principal para a construção de componentes dinâmicos de interface.
- **[TypeScript](https://www.typescriptlang.org/):** Tipagem estática para garantir maior confiabilidade e evitar erros comuns.
- **[Vite](https://vitejs.dev/):** Ferramenta de build rápida para empacotamento e servidor de desenvolvimento ágil.
- **[Tailwind CSS](https://tailwindcss.com/):** Framework CSS baseado em utilitários para desenvolvimento rápido de designs modernos.
- **[shadcn/ui](https://ui.shadcn.com/):** Biblioteca de componentes estilizados e acessíveis baseados em Radix UI.
- **[Bun](https://bun.sh/):** Utilizado como gerenciador de pacotes ultra-rápido para instalar dependências e rodar comandos.
- **[Lucide React](https://lucide.dev/):** Biblioteca de ícones bonitos e leves para interfaces web.

## ⚙️ Como executar o projeto localmente

Como o projeto utiliza o **Bun** como gerenciador principal, siga estes passos simples para rodá-lo na sua máquina:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/cargnieli2016/filhos-4-patas.git
   ```

2. **Navegue até a pasta do projeto:**
   ```bash
   cd filhos-4-patas
   ```

3. **Instale as dependências:**
   ```bash
   bun install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   bun run dev
   ```

5. **Acesse o site:**
   Abra o link exibido no seu terminal no navegador (geralmente [http://localhost:5173](http://localhost:5173)).
